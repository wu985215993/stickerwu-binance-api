## Description

[Nest](https://github.com/nestjs/nest) 基于 Nest 脚手架搭建的项目代码写的不好请提建议

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
# 生产环境通过 pm2 启动项目

$ pm2 start ecosystem.config.js --env production

```
